너는 세계 최고의 Next.js 개발자이자 UX 디자이너야. 
'MY PAGE' 우측에 버튼하나 만들어서 [출석 시스템]을 추가해줘. 자세한 기능은 아래에 명시할게. 
서버 없이 브라우저의 localStorage만 사용하여 모든 로직을 구현해야 해.

[출석시스템 핵심 로직: 이원화된 인증]
1. 타임슬롯: 오전(08~10시), 점심(11~13시), 저녁(17~19시) 하루 총 3회 적립 가능. [출근/점심시간/퇴근]
2. 일반 유저 (GPS 인증): 
   - [GPS 인증하기] 버튼 클릭 시 현재 좌표를 가져와 지정된 좌표(예: 한전 본사) 반경 100m 이내일 때만 10점 적립.
3. NFC 키링 유저 (하이패스):
   - URL 파라미터에 `?source=nfc`가 포함되어 접속할 경우, 위치 확인(GPS) 절차를 완전히 생략하고 즉시 출석 인정.
   - NFC 유저는 'NFC 보너스'를 포함하여 총 15점(기본 10 + 보너스 5)을 적립.

[데이터 관리 (localStorage)]
- 'checkInHistory' 키에 날짜별, 타임슬롯별 성공 여부를 저장하여 중복 적립을 방지해줘.
- 예: { "2026-01-16": { "morning": true, "lunch": false, "evening": false } }

[예술적 UI/UX 요구사항]
- 메인 대시보드 하단에 현재 출석 상태를 보여주는 스테이터스 카드를 배치해줘.
- NFC 접속 시: 
  - 화면에 "NFC 인증 성공!"이라는 화려한 텍스트 애니메이션과 함께 무지개빛 그라데이션 효과를 보여줘.
  - 모바일 진동 효과(navigator.vibrate)를 짧게 두 번(지잉, 지잉) 실행해줘.
- 캐릭터 반응: 서브타이틀 (참고 : '에너지수호자'->'신입사원'으로 명칭변경해줘)을 가진 유저 캐릭터가 말풍선으로 "출석 완료! 에너지 충전!"이라고 외치게 해줘.

[기술 사양]
- 프레임워크: Next.js (App Router), Tailwind CSS.
- 애니메이션: Framer Motion.
- 아이콘: lucide-react.
- 폰트/스타일: 기존의 Deep Navy 글래스모피즘 테마 유지.

단일 파일 `app/page.tsx` 또는 `/check-in` 페이지용 코드로 작성해줘.
🧠 리더를 위한 운영 꿀팁
NFC 키링 세팅: 키링에 쓸 URL은 꼭 https://사이트주소/check-in?source=nfc 형식으로 입력해둬야 해. 그래야 코드가 "아, 이 사람은 키링을 찍고 왔구나!"라고 바로 알아채거든.

GPS 좌표 설정: 클로드가 코드를 짜주면 그 안에 target_lat(위도)와 target_lng(경도)라는 변수가 있을 거야. 거기에 한전 본사나 팀장님이 계신 사무실 좌표를 넣으면 실제 위치 기반 인증이 작동하게 돼.

예술적 연출: NFC 유저가 들어올 때는 일반 유저와는 확연히 다른 '골드'나 '레인보우' 컬러의 이펙트를 주도록 설정했어. 옆 사람이 봤을 때 "우와, 저건 뭐예요?"라고 물어보게 만드는 게 우리의 전략이니까.
